# LMT SoM SDK

## Key Features

-  **nRF9160 Based**: Built on the powerful and versatile nRF9160, ensuring high performance and low power consumption.
-  **IoT Module**: Designed specifically for Internet of Things (IoT) applications, facilitating seamless connectivity and data exchange.
-  **Battery-Powered**: Optimized for extended battery life, crucial for remote and portable IoT devices.

-  **LTE-M Connectivity**: Utilizes LTE-M for reliable and efficient long-range wireless communication.

## Getting Started

To get started with the LMT SoM SDK, copy this folder to the your projects main folder and add 
    `add_subdirectory(lmtSDK)`
to your main CMakeLists.txt file.

## Supported Sensors

The Supported Sensors sensors list is backed by Zephyr.
