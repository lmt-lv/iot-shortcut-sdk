# lmtSDK

The latest lmtSDK. Contains the codebase for lmtSoM modules.

## Tools
- VS code
- nRF Connect SDK 2.6.0
- nRF Connect SDK Toolchain 2.6.0

## Installation
Add the lmtSoM boards definition folder to the nRF Connect SDK configuration using any method according to the https://academy.nordicsemi.com/courses/nrf-connect-sdk-intermediate/lessons/lesson-3-adding-custom-board-support/topic/exercise-1/ or copy the 'lmtSDK/boards' folder to the sample project folder 'boards'.

## Samples
- Check the samples under the samples folder opening the according folder in the VSC.
- Check the sample folder about it configuration.
- The project should be built using the lmt_som_nrf9160_ns board.
- In general all projects using lmtSDK should contain the child_image subfolder that can be copied from the lmtSDK folder.
- The project CMakeLists.txt should be valid.
